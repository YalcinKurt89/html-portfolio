What I like to work with in programming

HTML 🖊️

HTML is a markup language used for structuring and creating content on the web. It uses tags to define elements like headings, paragraphs, images, and links. Along with CSS for styling and JavaScript for interactivity, HTML forms the foundation of modern web development, allowing developers to create visually appealing and interactive websites.


CSS 🎨

CSS (Cascading Style Sheets) is a style sheet language used to control the visual appearance of HTML elements on a web page. It allows developers to define colors, fonts, spacing, and other styling properties. By separating the content from the presentation, CSS enables consistent and efficient design across multiple web pages.


Javascript 🎊

JavaScript is a programming language that enables interactivity and dynamic functionality on websites. It allows developers to create interactive elements, manipulate and modify content, and respond to user actions. JavaScript is widely used for client-side web development and is supported by all modern web browsers.
